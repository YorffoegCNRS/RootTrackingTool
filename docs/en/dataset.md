# Dataset

## Overview

The dataset associated with RootTrackingTool (RTT) is publicly available through the following DOI:

https://doi.org/10.57745/SW015R

This dataset was created to evaluate the performance of RTT and compare it with existing root image analysis software. It contains all images and measurements required to reproduce the analyses presented in the associated publication.


## Dataset Content

The repository contains:

* Raw scans of plants grown in rhizobox systems;
* Ilastik-processed images used as input for RTT;
* Raw scans of Arabidopsis thaliana plants grown in Petri dishes;
* Quantitative measurements used for software benchmarking;
* Reference measurements performed manually using ImageJ.



## Plant Species

The dataset includes images from several plant species:

* Arabidopsis thaliana
* Lettuce
* Maize
* Tomato

Images were acquired under different experimental conditions and at multiple developmental stages.

## Benchmark Data

The file `Software_comparison.xlsx` contains the measurements used for all software comparison analyses reported in the RTT publication.

It includes:

* Manual measurements obtained with ImageJ;
* Root number measurements;
* Total root length measurements;
* Measurements generated by:

  * RootTrackingTool (RTT)
  * RhizoVision Explorer
  * FarIA
  * RootSystemAnalyzer


## Reproducibility

This dataset provides all images and measurements necessary to reproduce the benchmarking analyses presented in the RTT publication.

It can also be used as a reference dataset for evaluating and comparing root image analysis software.
